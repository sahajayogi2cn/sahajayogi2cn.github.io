机器学习-3-神经网络

发表于 2018-04-17 | 分类于 机器学习

下面来总结下神经网络,是可以模拟任何函数的，具体原理可搜索”神经网络 任意函数”，这个和AND、OR等逻辑操作可以结合在一起思考，具体原理需要进一步学习研究呀!

那么如何训练数据呢，分如下几个步骤

  1. 随机初始化权重
  2. 实现前向网络 ,为每个样本 获得 h(Θ)
  3. 实现计算代价函数的代码 J(Θ)
  4. 对样本进行循环，对每个样本实现bp算法求偏导
  5. 使用梯度检验
  6. 使用梯度下降或者高级优化算法，得到最小化J(Θ)的Θ

这步是为了打破对称(Symmetry breaking), 假如有2个隐藏层1个输出层，INIT_EPSILON取值为0到1，那么初始化代码如下
