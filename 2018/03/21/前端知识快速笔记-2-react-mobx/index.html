发表于 2018-03-21 | 分类于 前端

MOBX

react-native mobx搭建(在现有项目里增加)
1.. 修改.babelrc形如 

{
 'presets': ['react-native'],
 'plugins': ['transform-decorators-legacy']
}

2.. package.json的dependencies加上 

"mobx": "^3.6.2",
"mobx-react": "^4.3.5",

package.json的devDependencies加上 

"babel-plugin-transform-decorators-legacy": "^1.3.4",

3.. 项目根目录下建个文件jsconfig.json，里面的内容为： 

{
    "compilerOptions": {
        "experimentalDecorators": true
    }
}

4.. npm install，大功告成，还可以使用装饰器啦

react native热更新

1.. 执行： (如果使用了enumify) 

npm install babel-preset-es2015 --save
npm install babel-preset-react --save
yarn

2.. 打包 

react-native bundle --entry-file index.android.js --bundle-output ./bundle/index.android.bundle --platform android --assets-dest ./bundle --dev false

react native 命令一览

1.. vscode里过滤android的log的tag 

adb logcat -s "cjx111"
